# Deal Tracker

Deal Tracker is a full-stack web application to manage, track, and monitor business deals securely with JWT authentication.

## Features
- Secure user authentication with JWT
- Add, update, view, and delete deals
- Dashboard overview of deals
- API-ready for integrations and extensions

## Technology Stack
- Backend: Java Spring Boot
- Database: MySQL / PostgreSQL
- Frontend: React.js
- Authentication: JWT (JSON Web Tokens)

## Installation
1. Clone the repository:
   ```bash
   git clone <your-repo-url>
      Navigate to the project folder and open in your IDE.
2. Configure database settings in application.properties.
3. Run the Spring Boot application.

## Usage
- Register a new user or login.
- Add deals using the dashboard.
- Track deal status and update information as needed.
