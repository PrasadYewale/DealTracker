# Deal Tracker

Deal Tracker is a full-stack web application to manage, track, and monitor business deals securely with JWT authentication.

## Features
- Secure user authentication with JWT
- Add, update, view, and delete deals
- Dashboard overview of deals
- API-ready for integrations and extensions

## Technology Stack
- Backend: Java Spring Boot
- Database: MySQL / PostgreSQL
- Frontend: React.js
- Authentication: JWT (JSON Web Tokens)
